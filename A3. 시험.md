# 시험(Test) 계획서

* 지침 *

- 시험은 자동 시험, 수동 시험으로 나눈다.
- 자동 시험을 할 수 있는 시험은 스스로 수행 한뒤 그 결과를 기록한다. 이후 시험에서 이전 자동 시험 항목 중 완료된 항목은 삭제 처리한다. 문제 있는 항목은 별도 섹션으로 보고서를 작성한다.
- 수동 시험 항목은 1. 수동 시험 항목 리스트, 2. 수동 시험 절차 및 판정 기준 순으로 정리한다.
- 수동 시험 항목중 완료된 것은 완료 시험 이력으로 옮긴다.
- 완료 시험 이력은 직전 버전의 수동 시험 완료 내용만 남기고 버전이 2개 이상 차이가 나면 시험-아카이브.md에 내용을 추가/누적시킨다.

## 현재 시험 항목

### v0.7.4 — Preferences 수정 및 About 모달 추가 수동 시험 항목

#### 수동 시험 항목 리스트
- **MT-7.4-1**: Top bar 메뉴 구조 검증 (Project, Edit, Help가 올바른 하위 메뉴 구성을 가졌는지 확인)
- **MT-7.4-2**: Preferences Window 기동 및 검은 화면 해결 검증 (Edit -> Preference (Setup) 클릭 시 검은 화면 없이 UI가 즉시 정상 표시되는지 확인)
- **MT-7.4-3**: Preferences Window 창 닫기 및 포커스 복원 (우상단 붉은 닫기 버튼 클릭 시 안전하게 닫히고 메인 윈도우가 정상 동작하는지 확인)
- **MT-7.4-4**: 테마 변경 및 실시간 동기화 검증 (Preferences 창에서 테마 카드 클릭 시 메인 윈도우와 Preferences 윈도우 양쪽 다 즉시 테마 CSS 변수가 동적 반영되고, 창들이 실시간 연동되는지 확인)
- **MT-7.4-5**: About Window 기동 및 내용 검증 (Help -> About 클릭 시 420x300 크기의 borderless modal 창이 기동되고 앱 로고, 이름, 버전(v0.7.4), 설명이 표시되는지 확인)
- **MT-7.4-6**: About Window 확인(Confirm) 및 닫기 검증 (우상단 X 또는 하단 "확인" 버튼 클릭 시 창이 닫히고 메인 윈도우가 정상 작동하는지 확인)

#### 수동 시험 절차 및 판정 기준
1. **MT-7.4-1: Top bar 메뉴 구조**
   - 절차: 앱을 기동한 뒤 top bar의 Project, Edit, Help 메뉴를 차례로 클릭한다.
   - 판정 기준:
     - Project: New Session, Open, Save Session, Import Files..., Import Folder, Quit 이 구분선과 함께 올바르게 나열된다.
     - Edit: Undo, Redo, Preference (Setup) 이 구분선과 함께 나열된다.
     - Help: Manual, Check for Updates..., About 이 구분선과 함께 나열된다 (About 옆에 현재 v0.7.4 버전 정보가 함께 표시된다).
     - 기존의 Theme 드롭다운이 top bar에서 완전히 보이지 않는다.
2. **MT-7.4-2: Preferences Window 기동 및 검은 화면 해결**
   - 절차: Edit -> Preference (Setup) 메뉴를 클릭한다.
   - 판정 기준: width 640, height 480 크기의 borderless 모달 자식 창이 즉시 뜨며, **더 이상 검은 화면이 아닌** 정상적인 환경설정 UI가 렌더링된다. 창을 드래그해 움직일 수 있고(타이틀 바 영역), 메인 윈도우 위에서 포커싱을 유지한다.
3. **MT-7.4-3: Preferences Window 닫기**
   - 절차: Preferences 창의 우상단에 위치한 닫기(X) 버튼에 마우스를 올린 뒤 클릭한다.
   - 판정 기준: 마우스 오버 시 X 버튼이 붉은색으로 하이라이트되며, 클릭 시 Preferences 창이 닫힌다. 메인 윈도우는 그대로 유지된다.
4. **MT-7.4-4: 테마 변경 및 실시간 동기화**
   - 절차: Preferences 창을 열고, "Color Theme" 탭에서 8가지 테마 카드(Teal, Sunset 등)를 차례로 클릭한다.
   - 판정 기준:
     - 각 테마 카드 내에 테마의 주요 토큰 색상을 대변하는 circular dots(Accent, Background 등)가 정상적으로 표시된다.
     - 테마 카드를 클릭하면 Preferences 창의 테마와 메인 윈도우의 테마가 동시에 즉시 업데이트된다.
     - 활성화된 테마 카드는 accent 색상의 테두리와 glow 효과 및 "Active" 뱃지가 표시된다.
5. **MT-7.4-5: About Window 기동 및 내용**
   - 절차: Help -> About 메뉴를 클릭한다.
   - 판정 기준: width 420, height 300 크기의 borderless 모달 자식 창이 뜬다. 창 내부에는 앱 로고, 앱 이름(FocusDAW - Mastering Desk), 버전 배지(Station Edition v0.7.4), 간략 설명글, 그리고 하단에 "확인" 버튼이 올바르게 렌더링된다.
6. **MT-7.4-6: About Window 확인(Confirm) 및 닫기**
   - 절차: About 창의 우상단 X 버튼 또는 하단의 "확인" 버튼을 클릭한다.
   - 판정 기준: 클릭 시 About 창이 안전하게 닫힌다. 메인 윈도우는 그대로 유지되며 포커스를 복원한다.
