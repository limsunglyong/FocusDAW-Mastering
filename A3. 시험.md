# 시험(Test) 계획서

* 지침 *

- 시험은 자동 시험, 수동 시험으로 나눈다.
- 자동 시험을 할 수 있는 시험은 스스로 수행 한뒤 그 결과를 기록한다. 이후 시험에서 이전 자동 시험 항목 중 완료된 항목은 삭제 처리한다. 문제 있는 항목은 별도 섹션으로 보고서를 작성한다.
- 수동 시험 항목은 1. 수동 시험 항목 리스트, 2. 수동 시험 절차 및 판정 기준 순으로 정리한다.
- 수동 시험 항목중 완료된 것은 완료 시험 이력으로 옮긴다.
- 완료 시험 이력은 직전 버전의 수동 시험 완료 내용만 남기고 버전이 2개 이상 차이가 나면 시험-아카이브.md에 내용을 추가/누적시킨다.

## 현재 시험 항목

### 1. 수동 시험 항목 리스트
- **MT-8.9-1**: Help 메뉴 Transport(F4) 우측 배치 및 드롭다운 시험
- **MT-8.9-2**: 섹션별 독립 Undo/Redo 파라미터 복구 시험
- **MT-8.9-3**: 노브/EQ 드래그 조작 시 Undo 스택 그룹화 시험
- **MT-8.9-4**: Export 진행 중 취소(Cancel) 동작 시험

### 2. 수동 시험 절차 및 판정 기준
- **MT-8.9-1 (Help 메뉴 배치)**:
  - 절차: 메인 앱 창의 TitleBar에서 Help 메뉴 버튼이 Transport(F4) 버튼 바로 우측에 위치해 있는지 확인하고, 클릭 시 드롭다운이 알맞게 아래로 열리는지 확인한다.
  - 판정 기준: Help 메뉴 버튼이 Transport(F4) 우측에 정확하게 정렬되어 있으며, 클릭 시 드롭다운 메뉴가 올바르게 표시된다.
- **MT-8.9-2 (섹션별 독립 Undo/Redo)**:
  - 절차: 
    1. II. Pre-Proc 화면에서 노이즈 리덕션 값을 수정한 뒤, III. EQ 화면으로 이동한다.
    2. III. EQ 화면에서 EQ Gain 값을 임의로 수정한 뒤, `Ctrl+Z`를 여러 번 입력한다.
    3. EQ Gain 값만 이전으로 롤백되는지 확인하고, 다시 II. Pre-Proc 화면으로 돌아갔을 때 노이즈 리덕션 값은 롤백되지 않고 유지되어 있는지 확인한다.
  - 판정 기준: 특정 설정창에서 Undo/Redo를 수행했을 때, 다른 설정창의 파라미터는 백그라운드에서 변경되지 않고 오직 현재 활성화된 화면의 파라미터만 안전하게 복구된다.
- **MT-8.9-3 (드래그 조작 시 스택 그룹화)**:
  - 절차:
    1. 임의의 노브나 EQ 그래프 노드를 마우스 드래그로 크게 흔들어 연속적으로 값을 변경한다.
    2. 드래그를 마친 후 `Ctrl+Z` 단축키를 한 번 입력한다.
  - 판정 기준: 여러 차례 값이 변했음에도 단 한 번의 Undo로 드래그를 시작하기 직전의 상태로 원복된다.
- **MT-8.9-4 (Export 진행 중 취소 및 UI 피드백)**:
  - 절차:
    1. 오디오 파일 여러 개를 로드한 후, VII. Export 화면에서 "EXPORT ALL" 버튼을 눌러 인코딩을 시작한다.
    2. 화면에 나타나는 EXPORTING 오버레이(LoadingCard) 하단의 "CANCEL EXPORT" 버튼을 클릭한다.
    3. 클릭 즉시 오버레이 제목이 "CANCELLING..."으로 변경되면서 깜박이는지 확인하고, 버튼이 비활성/숨김 처리되는지 확인한다.
    4. 내보내기가 취소된 후 메인 화면으로 복귀했을 때, "Export cancelled." 오류 메시지가 우측 설정 패널이 아닌 왼쪽 "ALBUM ARTWORK" 및 "DESTINATION" 영역 아래에 독립적인 별도 행으로 표시되는지 확인한다.
  - 판정 기준: 취소 클릭 시 로딩 타이틀이 "CANCELLING..."으로 전환되어 깜박이고, 완료 후 "Export cancelled." 안내가 왼쪽 하단 영역에 정상 노출된다.
