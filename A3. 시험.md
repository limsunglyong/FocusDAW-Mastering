# 시험(Test) 계획서

## 현재 시험 항목

### v0.2.28 — STFT Denoise 실처리 (수동 시험)

[시험 환경 준비]
- `npm run dev` 실행, **노이즈가 들리는 오디오**(예: hiss/hum 있는 녹음) 로드.

[시험 항목 / 방법 및 절차]
- MT-2A-1 Denoise 청감: 좌측 Play 로 재생 중 Pre(II)의 **Denoise 스위치 ON** → 잠깐 'Denoising' 오버레이 후 노이즈가 줄어 들리는지(현재 위치에서 끊김 최소).
- MT-2A-2 Noise Depth: Depth 1→2→3 으로 올릴수록 노이즈 제거가 강해지는지(3 Deep 가장 강함). 각 변경 시 재계산(오버레이) 후 반영.
- MT-2A-3 Noise Reduction(%): 노브를 올릴수록 제거 강도가 커지고, 0%면 사실상 원음인지. 드래그 중 과도한 재계산 없이 멈춘 뒤(약 0.35초) 반영되는지.
- MT-2A-4 Bypass/OFF 복귀: Pre 카드 Bypass 또는 Denoise OFF 시 원음(노이즈 포함)으로 즉시 돌아오는지.
- MT-2A-5 워터폴 연동: Denoise ON 시 3D 스펙트로그램과 Noise Floor/SNR 가 denoise 적용 기준으로 바뀌는지(고역 노이즈 바닥이 낮아짐).
- (자동) AT-2A-6 회귀: `npm run lint`·`npm run build`(60 modules)·`npm run verify`(34/34) 통과.
