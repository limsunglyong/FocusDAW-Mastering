# 지침

*중요*
1. 하단의 수정요청사항을 읽고 문제점을 파악한뒤 수정계획을 `앱개발.md`에 기록한다.
2. 앱 버전 관리 규칙은 `앱개발.md` 의 지침을 참고합니다.
3. 반드시 사용자에게 내용을 알리고 수정에 대한 승인을 받은 뒤 코드/문서 수정을 진행한다.
4. 수정이 완료되면 구현된 사항을 `앱개발.md` 에 기록하고 이에 대한 시험 방안을 쉽고 간단하게 [시험항목], [시험방법 및 절차], [시험 환경 준비] 등의 내용을 `시험.md` 에 내용을 추가하고 업데이트 한다.
5. 아래 수정요청사항의 각 항목에 대한 수정이 완료되면 항목을 삭제한다. 만약 어떤 이유로 수정을 하지 못했다면 그 이유와 함께 적어두고 혹시 추가 수정사항이 필요한 경우 해당 내용을 추가해둔다.


# 수정요청사항

1. 시험 결과

- MT-T1 : 기능 동작은 정상 확인하였음. 보완 요청 사항 : 현재는 전체 윈도우 사이즈를 고정하고 transport 패널을 열면 기존 항목들의 높이를 줄이는 방향인데, 이것을 전체 윈도우를 transport 패널 크기만큼 늘리고 기존 항목들의 높이는 그대로 유지하는 방향으로 수정 요청.

- MT-T2 : 정상 확인 완료.
- MT-T3 : 정상 확인 완료.
- MT-T4 : 정상 확인 완료.
- MT-T5 : 정상 확인 완료.
- MT-T6 : 정상 확인 완료.
- MT-T7 : 정상 확인 완료.

2. 기타

- Transport 메뉴 이름을 Transport(F4) 로 변경하고 단축키 F4 로 열고 닫을 수 있도록 수정 요청.

- Transport 패널에서 시간 표시 폰트 수정 요청 : 현재 play 시간 표시 폰트는 지금보다 2배 크게 변경. play 시 현재 시간은 하이라이트 색상으로 breathing 효과. pause시 breathing 효과 멈춤. stop시 하이라이트 -> normal 로 변경. 노래 전체 길이(시간) 표시 폰트는 지금보다 약 150% 증가. 이탤릭 체로 변경.

- volume 슬라이더 모양을 현재에서 큰 라운드형 직사각형 스타일로 변경. 
- volume 슬라이더를 변경하고 나서 space로 play/pause가 안되는 현상 보완/수정 요청.
