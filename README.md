FocusDAW – Mastering Desk
FocusDAW is a Windows desktop application designed for fast, consistent mastering across multiple audio tracks.
Key Features
WAV, MP3, FLAC, OGG, M4A, AIFF, and AIF input support
Denoise and fade processing
Min-Phase Parametric EQ and 9-Band Graphic EQ
Multiband dynamics and exciter
Stereo width, bass mono, reverb, and delay
Target LUFS control and true-peak limiting
User EQ presets and reusable Session Cards
Multi-job Render Batch with different Session Cards for each file group
WAV, MP3, and FLAC export
Built-in Korean and English manual
Automatic updates for Windows
Compare the original and processed audio through Preview, save your mastering settings as Session Cards, and apply them consistently to multiple files or folders.
